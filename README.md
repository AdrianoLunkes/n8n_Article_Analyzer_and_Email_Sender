# Article Analyzer and Email Sender (n8n + Gemini + Google Sheets + Gmail)
A scheduled n8n workflow that fetches English articles, uses Gemini 2.5 Flash to extract &amp; translate the top 5 key topics into Spanish, and automatically dispatches the summarized report via Gmail, pulling article links and recipient emails from Google Sheets.

# 1. Project Explanation

The goal of this project is to automate the analysis of two different articles on a scheduled basis and send their top 5 summarized topics—generated by Gemini 2.5 Flash—to a list of recipient emails.

This workflow is built using:

n8n (automation engine)

Google Sheets (data source for article links & email list)

Google Drive (storage)

Google AI Studio (Gemini API key generation)

Gemini 2.5 Flash (LLM for topic extraction)

HTML Extract (content parsing)

HTTP Request (article fetching)

Gmail (email dispatch)

In summary, the automation pulls article URLs from Google Sheets, fetches the article content, processes it with Gemini, and sends the summarized insights to email addresses defined in the same spreadsheet.

# 3. Setup: HTTP & HTML Parsing
Node: HTTP Request

Input: Article URL (from column “Links”)

Purpose: Download the HTML content of the article

The HTTP response is typically JSON-like, so:

Node: HTML Extract

Extract text content from <body> tags

Enable “Return Array” so all text elements of the article are captured

Create a field: txt_key

This field will store the extracted text that will be sent to Gemini

This step prepares the raw content for LLM analysis.

# 4. Creating an API Key from Gemini (Google AI Studio)

Using the same Google account connected to the sheets & Gmail:

Go to Google AI Studio

Create a new API key

Return to n8n → Add a new node:
Google Vertex AI / Gemini (Message Model)

Paste the API Key into the API Key field of the Gemini node

This node now becomes the LLM engine that will analyze the article text.

# 5. Gemini Prompt Setup

Inside the Gemini node, insert the custom prompt:
```
Prompt Used in the Project
Tu tarea es analizar el texto proporcionado y generar un resumen claro, preciso y estructurado en exactamente 5 puntos.

REQUISITOS OBLIGATORIOS:
1. El resumen debe estar SIEMPRE escrito en español (España), independientemente del idioma del texto original.
2. Debes generar exactamente 5 puntos, numerados del 1 al 5.
3. Cada punto debe ser breve pero informativo.
4. No añadas conclusiones adicionales, notas, explicaciones ni advertencias.
5. No traduzcas el texto completo, solo resume su contenido.

Formato de salida:
1. …
2. …
3. …
4. …
5. …

Texto para analizar:
{{ $json["txt_key"] }}
```

The variable {{ $json["txt_key"] }} contains the extracted article body text.

The output of this node is the 5 key summarized topics, in Spanish (Spain), ready for email distribution.

# 6. Implementing the Email Dispatch Step

After Gemini produces the 5 summarized topics:

New Node: Gmail – Send Email

Create a new Gmail credential

Log in with the same Google account used for Sheets and AI Studio

Now complete the form:

Email Configuration

To:

From the Google Sheets row (Column “Email”)

Subject:

Article Summaries Attached – {{ $json["Links"] }}


Email Type:

Text

Message Body:

Insert the output from the Gemini node (the 5 summarized topics)

The email will include:

The article link being analyzed

The summarized 5 topics extracted by Gemini

Sent to each email in the spreadsheet row
